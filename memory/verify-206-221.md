## 详细验收记录

### P0 安全类

#### #212 硬编码 Token 移除 ✅
- **方法：** 全局 grep 搜索 + 代码审查
- **结果：** 文档中 token 已替换为 `<agent-token>` 占位符；代码中 token 从 config.yaml 读取
- **建议：** config.yaml 包含实际 token 值被提交到仓库，建议加入 .gitignore 或用 .yaml.example 模式

#### #213 平台→Agent 通信认证 ✅
- **方法：** 代码审查
- **结果：** Flask `before_request` 中间件验证 `X-Agent-Token` header，/status 豁免
- 401 响应 + 错误信息

#### #214 subprocess 安全边界 ✅
- **方法：** 代码审查
- **结果：** 
  - `SCRIPT_MAP` 硬编码白名单：evalType → 脚本文件名
  - `os.path.realpath` 路径穿越校验
  - 未知 evalType 抛 ValueError
