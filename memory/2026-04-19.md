## Lessons Learned
1. **Spring AOP 自调用陷阱** — 同一 bean 内方法 A 调 B，即使 B 有 @Transactional 也不生效（代理被绕过）。解决：在实际需要事务的方法上直接加 @Transactional
2. **DataParallel 不适合小模型** — torch.nn.DataParallel 在小模型（<1M params）上容易死锁。scatter/gather 通信开销远大于计算本身。小模型直接单 GPU 跑
3. **Docker 端口映射 443 无 SSL = 隐形炸弹** — Chrome HTTPS-First 尝试连 443，TCP 连上但 TLS 握手失败，Chrome 无法 fallback 到 HTTP。不用 HTTPS 就别映射 443
4. **模块隔离并行持续有效** — 今天 4 轮并行（report pipeline / frontend UX / report meta / task cleanup），零冲突。关键是文件集合不重叠
5. **5 个 issue 打包 > 超时概率高** — 再次验证 3 个一组最稳定，4+ 风险增大
