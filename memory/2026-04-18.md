## Lessons Learned

- **Model 全限定名很重要** — routerhub provider 下的模型必须用 `routerhub/xxx` 前缀，不能省略 provider
- **Agent 超时 ≈ build/deploy 阶段** — 大多数超时发生在 mvn build 或 docker compose，代码改动本身已完成。主 session 应直接接管 build 而非盲目重跑
- **Git push 降级方案有效** — bare clone + bundle 可以从有网络的机器 relay push
- **因果链修复效果显著** — #496→#497→#499 串行修复，Token 安全 → 离线检测生效 → GPU 不再泄漏，一次性解决整条链
- **Agent Python 代码可以快速迭代** — 不需要 Maven 构建，改完重启 systemd 即可，效率远高于 Java 后端
