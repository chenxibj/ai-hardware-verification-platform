## 设计决策
- 集群注册 3 种方式: Agent 接入(推荐) / kubeconfig / 云 API，参考 Rancher 模式
- 资源池严格类型化 NODE_POOL / K8S_POOL，创建后不可变
- 调度引擎根据池类型自动选择调度方式（Agent 直连 vs K8s Job）
- 运维监控对接 Prometheus + Loki 体系
- kubeconfig AES-256-GCM 加密存储
