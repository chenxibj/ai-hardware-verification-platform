## Lessons Learned

- **及时发每日总结** — chenxi 23:50 提醒才发，应该在 23:00 自动触发（cron 或心跳）
- **Skill 安装走 GitHub 镜像更稳** — clawhub CLI 有限速，npx skills 有格式兼容问题，直接 git clone + ghfast.top 镜像最可靠
- **GPT5.4 模型无 API key** — sub-agent spawn 时如果指定了 GPT5.4 会直接失败，用默认 Opus 4.6 最稳。以后不指定非默认模型除非确认有 key
- **9 个 bug 打包一个 agent 13 分钟搞定** — 单 agent 处理同类 bug（输入校验/异常处理）效率很高，比拆多个 agent 更快
- **Bug 清扫日效率惊人** — 一天 26 个 issue 从创建到关闭，关键是 issue 描述详尽（含复现步骤+根因+修复建议），agent 拿到就能动手
