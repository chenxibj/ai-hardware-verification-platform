# 02-05 评测日志与数据管理

**模块**: 评测结果及资产管理（核心支撑模块）  
**优先级**: P1  
**预计工时**: 20 人日

---

## 功能概述

负责所有评测任务的日志、原始数据、中间数据的全生命周期管理，支撑问题排查、任务复盘、数据追溯。

---

## 功能点清单

### 02-05-001 日志采集范围

**描述**: 全面采集评测任务全流程日志

**验收标准**:
- [ ] 任务创建日志（创建人、创建时间、参数配置）
- [ ] 资源分配日志（分配的算力资源、分配时间、释放时间）
- [ ] 任务执行日志（执行步骤、执行状态、耗时、核心参数）
- [ ] 异常日志（异常类型、异常时间、异常原因、报错信息）
- [ ] 任务终止/重试日志（操作人、操作时间、操作原因）

**优先级**: P0

---

### 02-05-002 采集频率

**描述**: 实时采集任务执行过程中的日志

**验收标准**:
- [ ] 默认每 10 秒采集一次
- [ ] 高优先级任务每 2 秒采集一次
- [ ] 采集频率可配置
- [ ] 确保日志的实时性与完整性

**优先级**: P1

---

### 02-05-003 日志存储

**描述**: 按规范目录结构分类存储日志

**验收标准**:
- [ ] 按"租户 ID-任务 ID-日志类型 - 时间戳"目录结构存储
- [ ] 支持分布式存储
- [ ] 日志存储期限可配置（默认保留 1 年，核心任务日志保留 3 年）
- [ ] 支持日志压缩存储

**优先级**: P0

---

### 02-05-004 日志格式

**描述**: 统一日志格式

**验收标准**:
- [ ] 日志时间、日志级别（INFO/WARN/ERROR）
- [ ] 任务 ID、租户 ID、操作人
- [ ] 日志内容、关联资源 ID
- [ ] 确保日志可解析、可追溯

**优先级**: P1

---

### 02-05-005 多条件检索

**描述**: 支持多条件组合检索日志

**验收标准**:
- [ ] 按任务 ID、租户 ID、日志级别、时间范围、关键词检索
- [ ] 检索结果可按时间升序/降序排序
- [ ] 快速定位目标日志

**优先级**: P1

---

### 02-05-006 日志查看

**描述**: 支持在线查看日志详情

**验收标准**:
- [ ] 按任务执行流程顺序展示日志
- [ ] 支持日志分页
- [ ] 关键词高亮
- [ ] 支持日志过滤（仅显示 ERROR 级异常日志）

**优先级**: P1

---

### 02-05-007 日志下载

**描述**: 支持将日志批量下载

**验收标准**:
- [ ] 支持 TXT/CSV 格式
- [ ] 可选择下载全部日志或指定时间范围的日志
- [ ] 下载的日志保留原始格式

**优先级**: P1

---

### 02-05-008 评测数据分类

**描述**: 将评测数据分为三类管理

**验收标准**:
- [ ] 原始数据（用户上传的评测对象、数据集）
- [ ] 中间数据（评测过程中生成的临时数据、中间计算结果）
- [ ] 结果数据（最终评测指标、评测报告核心数据）

**优先级**: P1

---

### 02-05-009 数据存储

**描述**: 评测数据的存储管理

**验收标准**:
- [ ] 原始数据、结果数据保留期限与对应评测报告一致
- [ ] 中间数据默认保留 7 天
- [ ] 敏感数据自动脱敏存储

**优先级**: P1

---

### 02-05-010 数据关联

**描述**: 评测数据与任务、报告关联

**验收标准**:
- [ ] 通过任务 ID 可快速查询对应的原始数据、中间数据、结果数据
- [ ] 实现数据全链路追溯

**优先级**: P1

---

### 02-05-011 数据导出

**描述**: 支持用户导出评测数据

**验收标准**:
- [ ] 支持导出原始数据、结果数据
- [ ] 中间数据不支持导出
- [ ] 导出格式与数据类型匹配
- [ ] 导出时需验证用户权限

**优先级**: P1

---

### 02-05-012 数据加密

**描述**: 评测数据的安全加密

**验收标准**:
- [ ] 传输过程中采用 HTTPS 加密
- [ ] 存储过程中采用 AES-256 加密
- [ ] 敏感数据采用脱敏存储

**优先级**: P0

---

### 02-05-013 数据访问控制

**描述**: 按用户/角色、租户划分数据访问权限

**验收标准**:
- [ ] 用户仅可访问个人/租户创建的评测数据
- [ ] 管理员可访问全平台数据（需记录访问日志）
- [ ] 禁止跨租户访问数据

**优先级**: P0

---

### 02-05-014 合规审计

**描述**: 所有数据操作均记录审计日志

**验收标准**:
- [ ] 记录操作人、操作时间、操作内容、数据 ID、操作结果
- [ ] 审计日志保留 3 年
- [ ] 便于合规检查与责任追溯

**优先级**: P1

---

### 02-05-015 数据清理

**描述**: 支持用户清理无效评测数据

**验收标准**:
- [ ] 手动清理个人/租户的无效评测数据
- [ ] 支持批量清理
- [ ] 清理前二次确认
- [ ] 清理后不可恢复

**优先级**: P2

---

## 依赖关系

- 依赖模块 1-01（评测任务管理）提供日志采集
- 依赖模块 5-02（资源池管理）提供存储资源

## 接口需求

- 日志采集 API
- 日志检索 API
- 数据管理 API
