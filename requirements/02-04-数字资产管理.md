# 02-04 数字资产管理

**模块**: 评测结果及资产管理（核心支撑模块）  
**优先级**: P0  
**预计工时**: 30 人日

---

## 功能概述

负责平台内所有数字资产（模型、数据集、算子、脚本、流程模板等）的全生命周期管理。**本模块所有资产管理服务按存储容量、使用时长等资源消耗收费。**

---

## 功能点清单

### 02-04-001 资产分类管理

**描述**: 按资产类型进行分类管理

**验收标准**:
- [ ] 五大类：模型资产、数据集资产、算子资产、脚本资产、流程模板资产
- [ ] 每大类下按细分类型进一步分类
- [ ] 支持管理员自定义资产分类维度

**优先级**: P0

---

### 02-04-002 资产标签

**描述**: 支持为资产添加自定义标签

**验收标准**:
- [ ] 标签可批量添加、修改、删除
- [ ] 便于资产检索与筛选
- [ ] 标签与评测类型标签联动

**优先级**: P1

---

### 02-04-003 多类型资产上传

**描述**: 支持多类型资产上传

**验收标准**:
- [ ] 模型：ONNX、PyTorch、TensorFlow 等主流格式
- [ ] 数据集：Excel、CSV、图片文件夹、文本文件等
- [ ] 算子：Python/C++ 实现文件
- [ ] 脚本：Python 脚本
- [ ] 流程模板：JSON 格式
- [ ] 支持单个上传与批量上传

**优先级**: P0

---

### 02-04-004 资产信息填写

**描述**: 上传时需填写资产基础信息

**验收标准**:
- [ ] 资产名称、描述、版本号
- [ ] 分类、标签、适用场景
- [ ] 依赖环境
- [ ] 必填项缺失时无法完成上传

**优先级**: P0

---

### 02-04-005 自动校验

**描述**: 上传后自动校验资产

**验收标准**:
- [ ] 完整性校验
- [ ] 兼容性校验
- [ ] 合规性校验
- [ ] 校验失败时明确提示失败原因及修改建议

**优先级**: P0

---

### 02-04-006 资产版本管理

**描述**: 支持资产的多版本管理

**验收标准**:
- [ ] 自动生成版本号（v1.0、v1.1、v1.2）
- [ ] 支持用户手动指定版本号
- [ ] 记录每版本资产的修改历史

**优先级**: P1

---

### 02-04-007 版本追溯

**描述**: 记录资产的版本历史

**验收标准**:
- [ ] 修改人、修改时间、修改内容
- [ ] 版本说明
- [ ] 支持查看各版本详情
- [ ] 对比不同版本的差异

**优先级**: P1

---

### 02-04-008 版本回滚

**描述**: 支持资产版本回滚

**验收标准**:
- [ ] 回滚至历史任意版本
- [ ] 回滚前弹出二次确认提示
- [ ] 原当前版本仍保留在版本列表中

**优先级**: P1

---

### 02-04-009 版本锁定

**描述**: 支持对核心版本进行锁定

**验收标准**:
- [ ] 锁定后无法修改、删除该版本
- [ ] 仅可查看、复用
- [ ] 防止误操作导致核心资产丢失

**优先级**: P2

---

### 02-04-010 多条件检索

**描述**: 支持多条件组合检索资产

**验收标准**:
- [ ] 按资产名称、分类、标签、适用场景、创建人、版本号检索
- [ ] 支持模糊检索
- [ ] 检索结果可按相关性、创建时间、使用频率排序

**优先级**: P1

---

### 02-04-011 资产预览

**描述**: 支持资产在线预览

**验收标准**:
- [ ] 数据集预览（查看前 100 条数据、数据结构、字段说明）
- [ ] 流程模板预览（可视化展示流程节点及依赖关系）
- [ ] 脚本预览（在线查看脚本代码，不允许编辑）

**优先级**: P1

---

### 02-04-012 资产复用

**描述**: 支持用户直接复用平台内的资产

**验收标准**:
- [ ] 可选择任意历史版本
- [ ] 复用后自动记录复用记录
- [ ] 复用用户、复用场景、复用时间

**优先级**: P1

---

### 02-04-013 复用统计

**描述**: 自动统计各资产的复用情况

**验收标准**:
- [ ] 复用次数统计
- [ ] 复用场景统计
- [ ] 生成资产复用排行榜

**优先级**: P2

---

### 02-04-014 资产分享与权限控制

**描述**: 支持资产分享与权限配置

**验收标准**:
- [ ] 三种分享范围：个人使用、团队共享、全平台共享
- [ ] 精细化权限配置（查看、复用、编辑、下载）
- [ ] 分享追溯、分享撤销

**优先级**: P1

---

### 02-04-015 资产清理与备份

**描述**: 支持资产清理与备份

**验收标准**:
- [ ] 手动删除无效资产（回收站保留 30 天）
- [ ] 自动备份（每日凌晨备份）
- [ ] 手动备份
- [ ] 存储监控

**优先级**: P1

---
