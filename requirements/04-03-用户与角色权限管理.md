# 04-03 用户与角色权限管理

**模块**: 用户体系（基础支撑模块）  
**优先级**: P0  
**预计工时**: 25 人日

---

## 功能概述

采用"角色权限分离（RBAC）"模式，实现用户、角色、权限的精细化配置，确保不同用户仅能操作对应权限范围内的功能。

---

## 功能点清单

### 04-03-001 系统预置角色

**描述**: 按用户类型、操作场景预置六大类核心角色

**验收标准**:
- [ ] 超级管理员：拥有平台全部操作权限
- [ ] 平台管理员：负责平台日常运营、内容审核、用户管理
- [ ] 租户管理员：负责租户内部管理
- [ ] 评测专员：负责创建、执行、监控评测任务
- [ ] 资产管理员：负责租户内数字资产的上传、管理、分享
- [ ] 普通用户：拥有基础操作权限

**优先级**: P0

---

### 04-03-002 自定义角色

**描述**: 支持管理员、租户管理员自定义角色

**验收标准**:
- [ ] 根据实际需求配置角色权限
- [ ] 自定义角色可复制、修改、删除
- [ ] 便于权限复用

**优先级**: P1

---

### 04-03-003 权限配置

**描述**: 权限按模块、功能点精细化划分

**验收标准**:
- [ ] 评测系统的"任务创建""任务监控""报告导出"分别对应不同权限
- [ ] 支持为角色批量分配、移除权限
- [ ] 权限配置实时生效

**优先级**: P0

---

### 04-03-004 角色分配

**描述**: 管理员可为平台用户分配角色

**验收标准**:
- [ ] 分配系统预置角色、自定义角色
- [ ] 租户管理员可为本租户内成员分配角色

**优先级**: P0

---

### 04-03-005 多角色关联

**描述**: 支持为单个用户分配多个角色

**验收标准**:
- [ ] 用户权限为多个角色权限的合集
- [ ] 可设置角色优先级
- [ ] 当角色权限冲突时，按高优先级角色权限执行

**优先级**: P1

---

### 04-03-006 权限继承

**描述**: 子角色可继承父角色的全部权限

**验收标准**:
- [ ] 可添加额外权限
- [ ] 便于角色层级管理

**优先级**: P2

---

### 04-03-007 权限变更

**描述**: 支持修改用户关联的角色

**验收标准**:
- [ ] 变更后实时更新用户权限
- [ ] 权限变更记录审计日志
- [ ] 包含操作人、操作时间、用户 ID、角色变更内容

**优先级**: P1

---

### 04-03-008 操作审计

**描述**: 所有用户的操作均记录审计日志

**验收标准**:
- [ ] 包含操作人、操作时间、操作内容、操作 IP、操作结果
- [ ] 审计日志保留 3 年
- [ ] 支持多条件检索、导出

**优先级**: P0

---

### 04-03-009 权限监控

**描述**: 实时监控用户权限使用情况

**验收标准**:
- [ ] 识别异常权限操作
- [ ] 触发安全告警
- [ ] 管理员可查看告警详情、处理异常

**优先级**: P1

---

### 04-03-010 权限回收

**描述**: 管理员可手动回收用户的部分/全部权限

**验收标准**:
- [ ] 用户离职、退出租户时，自动回收该用户对应的租户权限、角色权限
- [ ] 角色删除时，自动解除所有关联用户的该角色权限

**优先级**: P1

---

### 04-03-011 最小权限原则

**描述**: 引导管理员按"最小权限"为用户分配角色

**验收标准**:
- [ ] 仅授予用户完成工作所需的必要权限
- [ ] 降低权限滥用风险

**优先级**: P1
