# 05-02 资源池管理

**模块**: 异构资源纳管（基础支撑模块）  
**优先级**: P1  
**预计工时**: 25 人日

---

## 功能概述

将接入的异构资源整合为统一的资源池，实现资源池的分类、分区、配额管理。

---

## 功能点清单

### 05-02-001 资源池分类

**描述**: 按资源类型、归属、评测场景进行分类

**验收标准**:
- [ ] 按资源类型：算力资源池、存储资源池、网络资源池
- [ ] 按资源归属：平台自有资源池、合作云资源池、用户共享资源池
- [ ] 按评测场景：芯片评测资源池、模型评测资源池、分布式评测资源池

**优先级**: P0

---

### 05-02-002 资源分区

**描述**: 在资源池内按租户、任务优先级进行分区管理

**验收标准**:
- [ ] 为不同租户、不同优先级任务分配独立的资源分区
- [ ] 避免资源竞争
- [ ] 支持自定义资源分区（如为核心租户创建专属资源分区）

**优先级**: P1

---

### 05-02-003 资源池联动

**描述**: 不同资源池之间支持联动调度

**验收标准**:
- [ ] 算力资源池与存储资源池联动
- [ ] 确保评测任务的算力与存储资源匹配
- [ ] 支持资源池动态扩容、缩容

**优先级**: P1

---

### 05-02-004 资源池监控

**描述**: 实时监控各资源池的资源使用情况

**验收标准**:
- [ ] 资源使用率、任务执行情况、资源状态
- [ ] 生成资源池运行报表
- [ ] 管理员可查看资源池运行状态

**优先级**: P1

---

### 05-02-005 配额分配

**描述**: 按租户、用户等级分配资源配额

**验收标准**:
- [ ] 算力配额（GPU/CPU 使用时长、并发任务数）
- [ ] 存储配额（存储容量、数据保留期限）
- [ ] 网络配额（带宽限制）
- [ ] 不同租户、用户等级对应不同的配额标准

**优先级**: P0

---

### 05-02-006 配额调整

**描述**: 支持管理员手动调整资源配额

**验收标准**:
- [ ] 调整后实时生效
- [ ] 支持租户、用户申请配额扩容
- [ ] 提交扩容申请后，管理员审核（审核时限≤24 小时）

**优先级**: P1

---

### 05-02-007 配额监控

**描述**: 实时监控租户、用户的资源配额使用情况

**验收标准**:
- [ ] 配额使用达到 80% 时触发提醒
- [ ] 配额用尽时限制用户创建新的评测任务
- [ ] 提示用户申请扩容

**优先级**: P1

---

### 05-02-008 配额回收

**描述**: 租户过期、用户注销时自动回收资源配额

**验收标准**:
- [ ] 重新纳入资源池
- [ ] 供其他用户、租户使用
- [ ] 支持管理员手动回收闲置租户、用户的资源配额

**优先级**: P1

---

### 05-02-009 资源标签

**描述**: 为资源池内的每台资源、每个分区添加自定义标签

**验收标准**:
- [ ] 标签可批量添加、修改、删除
- [ ] 如"华为昇腾 GPU""FP16 精度""工业场景专用"

**优先级**: P1

---

### 05-02-010 标签联动

**描述**: 资源标签与评测任务标签、数字资产标签联动

**验收标准**:
- [ ] 便于资源精准匹配
- [ ] 如为芯片评测任务自动匹配"芯片评测资源池""国产 GPU"标签的资源

**优先级**: P1

---

### 05-02-011 资源检索

**描述**: 支持按多条件组合检索资源

**验收标准**:
- [ ] 按资源类型、规格参数、标签、归属、状态检索
- [ ] 支持模糊检索
- [ ] 快速定位所需资源

**优先级**: P1

---

### 05-02-012 资源排序

**描述**: 检索结果可按多种维度排序

**验收标准**:
- [ ] 按资源使用率、性能、接入时间、归属排序
- [ ] 便于管理员、用户选择合适的资源

**优先级**: P2

---
