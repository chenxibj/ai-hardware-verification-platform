# 01-01 评测任务管理

**模块**: 评测系统（核心业务模块）  
**优先级**: P0  
**预计工时**: 40 人日

---

## 功能概述

负责评测任务的全生命周期管理，从任务创建、调度、监控到终止/重试，确保任务高效、稳定执行。

---

## 功能点清单

### 01-01-001 模板化创建任务

**描述**: 预置常用评测模板，用户可直接选择模板快速创建任务

**验收标准**:
- [ ] 预置至少 3 种模板（芯片性能评测、模型精度评测、框架兼容性评测）
- [ ] 用户选择模板后可修改参数（数据集、硬件规格等）
- [ ] 模板参数校验通过后可成功创建任务

**优先级**: P0

---

### 01-01-002 自定义创建任务

**描述**: 支持用户灵活配置评测参数，满足个性化评测需求

**验收标准**:
- [ ] 支持配置评测类型（性能/精度/兼容性/稳定性）
- [ ] 支持配置评测对象（芯片/算子/框架/模型）
- [ ] 支持选择数据集、硬件资源规格、超时时间
- [ ] 支持自定义评测指标
- [ ] 参数校验失败时给出明确提示

**优先级**: P0

---

### 01-01-003 资源/工具选择

**描述**: 用户选择评测所需的数据集、评测工具、硬件资源

**验收标准**:
- [ ] 支持预置资源选择（平台提供的标准化数据集、工具、国产芯片/框架）
- [ ] 支持自定义资源上传（自有数据集、工具、框架）
- [ ] 自定义资源上传后触发自定义评测任务标识
- [ ] 平台未预置芯片自动触发合作云厂商资源联动

**优先级**: P0

---

### 01-01-004 任务校验

**描述**: 创建任务时自动校验参数完整性、合理性

**验收标准**:
- [ ] 自动校验硬件资源可用性
- [ ] 自动校验数据集合规性
- [ ] 校验失败给出明确提示和修改建议
- [ ] 未预置芯片自动检索云厂商可用资源并匹配

**优先级**: P0

---

### 01-01-005 任务类型判定与收费跳转

**描述**: 任务校验通过后，自动判断是否为自定义评测任务并跳转收费结算

**验收标准**:
- [ ] 自动识别自定义评测任务（上传自有资源或选择未预置参数）
- [ ] 自定义任务自动跳转至收费结算页面
- [ ] 预置任务直接进入调度队列

**优先级**: P0

---

### 01-01-006 优先级调度

**描述**: 支持高、中、低三级任务优先级设置

**验收标准**:
- [ ] 用户创建任务时可选择优先级（高/中/低）
- [ ] 高优先级任务优先占用算力资源
- [ ] 优先级调度策略可配置

**优先级**: P1

---

### 01-01-007 资源自动分配

**描述**: 根据任务硬件需求自动匹配算力资源池中的空闲资源

**验收标准**:
- [ ] 根据 GPU 型号、数量自动匹配资源
- [ ] 资源分配过程透明可追溯
- [ ] 用户可查看资源分配详情

**优先级**: P0

---

### 01-01-008 手动资源指定

**描述**: 支持用户手动指定算力资源池、具体硬件设备

**验收标准**:
- [ ] 用户可选择具体算力资源池
- [ ] 用户可指定具体硬件设备（如某几台国产 GPU）
- [ ] 手动指定优先级高于自动分配

**优先级**: P1

---

### 01-01-009 并发控制

**描述**: 限制单用户/租户的并发任务数

**验收标准**:
- [ ] 可按用户等级配置并发上限
- [ ] 可按租户类型配置并发上限
- [ ] 超出并发限制时给出明确提示

**优先级**: P1

---

### 01-01-010 任务排队机制

**描述**: 算力资源不足时任务自动进入排队队列

**验收标准**:
- [ ] 按优先级 + 创建时间排序
- [ ] 资源释放后自动触发排队任务执行
- [ ] 用户可查看排队顺序、预计执行时间

**优先级**: P1

---

### 01-01-011 实时进度展示

**描述**: 清晰展示每个任务的当前状态和进度

**验收标准**:
- [ ] 展示任务状态（排队中/运行中/完成/失败/终止）
- [ ] 展示完成百分比、已用时间、剩余时间
- [ ] 状态更新延迟≤5 秒

**优先级**: P0

---

### 01-01-012 详细日志查看

**描述**: 实时输出评测过程中的详细日志

**验收标准**:
- [ ] 实时输出参数配置、资源占用、执行步骤、异常信息
- [ ] 支持按关键词过滤日志
- [ ] 支持按时间范围筛选日志
- [ ] 支持日志下载（TXT 格式）

**优先级**: P0

---

### 01-01-013 资源占用监控

**描述**: 实时展示任务占用的算力资源

**验收标准**:
- [ ] 展示 GPU/CPU 利用率、内存占用、网络带宽
- [ ] 支持可视化图表（折线图、柱状图）
- [ ] 数据更新频率≤10 秒

**优先级**: P1

---

### 01-01-014 异常告警

**描述**: 任务出现异常时自动触发告警

**验收标准**:
- [ ] 支持失败、资源不足、超时、硬件故障等异常检测
- [ ] 告警方式支持邮件、站内信
- [ ] 用户可自定义告警接收方式
- [ ] 告警信息包含任务 ID、异常类型、原因、处理建议

**优先级**: P0

---

### 01-01-015 手动终止任务

**描述**: 支持用户手动终止运行中的任务

**验收标准**:
- [ ] 终止前弹出二次确认提示
- [ ] 终止后释放占用的算力资源
- [ ] 任务状态更新为"终止"
- [ ] 自定义评测任务按实际资源消耗结算，退还未使用部分费用

**优先级**: P1

---

### 01-01-016 自动重试

**描述**: 失败任务支持自动重试

**验收标准**:
- [ ] 用户可配置重试次数（1-5 次）
- [ ] 用户可配置重试间隔（10 分钟 -1 小时）
- [ ] 重试失败后状态更新为"失败"并触发告警

**优先级**: P1

---

### 01-01-017 手动重试

**描述**: 支持用户手动触发失败/终止任务的重试

**验收标准**:
- [ ] 重试时可修改部分任务参数（硬件资源、超时时间）
- [ ] 重试过程与正常任务一致

**优先级**: P2

---

### 01-01-018 操作追溯

**描述**: 所有任务终止、重试操作均记录审计日志

**验收标准**:
- [ ] 记录操作人、操作时间、操作内容
- [ ] 记录任务 ID、操作结果
- [ ] 审计日志可查询、可导出

**优先级**: P1

---

## 依赖关系

- 依赖模块 5（异构资源纳管）提供算力资源
- 依赖模块 4（用户体系）进行权限校验
- 为模块 2（评测结果及资产管理）提供评测数据

## 接口需求

- 资源分配 API
- 任务状态查询 API
- 日志采集 API
- 告警通知 API
